# 🏥 Hospital Management System (RDBMS)

## 📌 Project Overview
The Hospital Management System is a relational database project developed using SQL and RDBMS concepts. The project focuses on designing a structured database to efficiently manage hospital operations through well-defined tables and SQL queries.

---

## 🎯 Objectives
- Design a relational database for a hospital.
- Create multiple interconnected database tables.
- Implement Primary Key and Foreign Key relationships.
- Perform efficient data retrieval using SQL queries.

---

## ✨ Features
- 12 Database Tables
- Relational Database Design
- Primary Key & Foreign Key Relationships
- 20 SQL Queries
- Well-Structured Database Schema
